# Installation
1. Clone the repo:
    > git clone https://github.com/fftdufhudd/nvim.git /your/local/directory
2. Install dependencies:
    * chmod +x install.sh && ./install.sh
3. Manual steps:
    * Odin: donwload from [odin-lang.org](https://odin-lang.org/docs/install/) and ensure the path in **plugin/lsp.lua** matches your Installation
    * C#: Install [.NET SDK](https://dotnet.microsoft.com/en-us/download) from Microsoft
